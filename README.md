# Slot Machine 🎰

シンプルなスロットマシンゲームの実装プロジェクトです。  
リールの回転・停止、当たり判定、スコア / コイン管理など、スロットゲームに必要な基本要素をひと通り含んでいます。

---

## 📌 概要

- 3 リール（または任意本数）のスロットマシンゲーム
- ボタン操作で「BET」「SPIN」「RESET」などの挙動
- 当たりパターンに応じた配当（コイン増減）
- ゲームオーバー条件（コインが 0 になる、など）
- デバッグしやすいシンプルなコード構成

---

## 🎮 主な機能

- **スピン機能**  
  - ボタン（またはキー入力）でスロットを回転
  - 一定時間経過 or 入力で停止
- **役判定 / 配当計算**  
  - リールの出目をチェックして役を判定
  - 役ごとに設定された配当を加算
- **コイン / スコア管理**
  - BET 金額の減算
  - 当たり時のコイン増減
  - コインが 0 になったときのゲームオーバー処理
- **簡易 UI**  
  - 現在のコイン数
  - 直近の結果（役・配当）
  - メッセージ表示（例：`BIG WIN!` / `GAME OVER`）


---
